# Google Map 熱區圖範例

## Demo site
- https://town-intelligent-alpha.github.io/heat_map/

## 功能

#### 前端:
- 啟動時，獲取 GPS 權限
- 每隔固定時間獲取一次 GPS
- 將座標傳遞至後端

#### 後端:
- API
  - upload_gps
  - get_gps
